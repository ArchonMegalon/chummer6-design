# Chummer ADR Index

Approved architecture decisions for cross-repo Chummer design live here.

## Accepted
- [ADR-0001](/docker/chummercomplete/chummer-design/products/chummer/adrs/ADR-0001-contract-plane-canon.md): Contract-plane canon is package-owned and design-first.
- [ADR-0002](/docker/chummercomplete/chummer-design/products/chummer/adrs/ADR-0002-play-split-ownership.md): `chummer-play` owns the play shell and depends only on approved shared packages.
- [ADR-0003](/docker/chummercomplete/chummer-design/products/chummer/adrs/ADR-0003-ui-kit-split.md): `chummer-ui-kit` is the package-only shared UI boundary.
- [ADR-0004](/docker/chummercomplete/chummer-design/products/chummer/adrs/ADR-0004-hub-registry-split.md): `chummer-hub-registry` owns registry and publication state after extraction from run-services.
